📖 Project Story

The Idea

Imagine a system like Genshin Impact — you pull random characters using "Gems" and with each pull there's a chance to get a legendary character ✨

Goal: Build this system from scratch using pure Java with no external libraries, focusing on professional OOP principles

The Challenge

Build a complete system including: a random summon engine, pricing system, TCP server, Swing client, and animated effects

All in 47 Java files across 3 architectural layers with full separation of concerns
